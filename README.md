# ng-seed
> angular项目基础框架，解决了前后端分离后，前端代码打包合并、资源文件CDN分离部署、单点登录、密码加密、服务接口跨域、SPA页面SEO等问题    

## Usage

安装依赖包  
```bash
$ yarn install
# or 
$ yarn
```

开发环境  
```bash
$ yarn run dev
```

生产环境  
```bash
$ yarn run build
# then
$ yarn run start
```
